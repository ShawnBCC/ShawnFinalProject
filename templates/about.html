About Miniature Warrior

Miniature Warrior was created out of a passion for history and craftsmanship. The site focuses on miniature samurai figures, armor replicas, and historically inspired collectibles.

The database part of the application keeps track of each samurai figure, including name, clan, material, size, price, quantity, description, and image file.

The project demonstrates how a Flask web application connects to a MySQL database using mysql.connector.
